# Venus Mc Tools (VMTools)

一个集成各种实用工具的 Minecraft 1.21.11 Fabric 模组。

## ✨ 功能特性

### 🎯 路径点管理
- **快捷键打开**：默认按 `M` 键打开路径点管理界面
- **分组管理**：支持创建、编辑、删除分组（右键菜单操作）
- **路径点 CRUD**：添加、编辑、删除、复制路径点（右键菜单操作）
- **分组选择**：编辑路径点时可切换所属分组
- **颜色标记**：每个路径点和分组可选择不同颜色
- **快速传送**：每个路径点旁边有传送按钮，一键执行命令
- **搜索过滤**：按名称或命令搜索路径点
- **Toast 通知**：操作反馈提示（成功/失败）

### 📥 导入导出
- **导出**：将路径点导出为 JSON 文件，自动打开导出目录
- **导入**：弹出系统文件选择窗口，直接选择 JSON 文件导入
- **导入预览**：导入前显示分组和路径点数量预览
- **导入模式**：支持追加导入和替换全部两种模式

### ⌨️ 快捷键系统
- 每个工具独立绑定快捷键
- 支持自定义修改
- 默认快捷键：
  - `M` - 打开路径点管理

## 📦 安装

1. 确保已安装 [Fabric Loader](https://fabricmc.net/)（版本 0.16.0+）
2. 下载 [Fabric API](https://modrinth.com/mod/fabric-api) 并放入 `mods` 文件夹
3. 下载 VMTools 模组文件并放入 `mods` 文件夹
4. 启动游戏

## 🎮 使用方法

### 添加路径点
1. 按 `M` 键打开路径点管理界面
2. 点击底部的 "+ 路径点" 按钮
3. 输入名称和传送命令（如 `/res tp main_city`）
4. 选择所属分组和颜色
5. 点击 "添加路径点" 保存

### 添加分组
1. 按 `M` 键打开路径点管理界面
2. 点击底部的 "+ 分组" 按钮
3. 输入分组名称
4. 选择颜色
5. 点击 "添加分组" 保存

### 传送
1. 按 `M` 键打开路径点管理界面
2. 展开分组，找到目标路径点
3. 点击路径点名称或右侧的 "传送" 按钮
4. 客户端会自动发送命令到服务器

### 右键菜单
- **右键分组**：展开/折叠、编辑、删除
- **右键路径点**：传送、编辑、复制、删除
- 删除操作需要二次确认，防止误触

### 导入导出
- **导出**：点击 "导出" 按钮，自动打开导出目录
- **导入**：点击 "导入" 按钮，弹出文件选择窗口选择 JSON 文件

## 📁 配置文件位置

```
.minecraft/
└── config/
    └── vmtools/
        ├── waypoints.json          # 路径点数据（主文件）
        ├── waypoints.backup.json   # 自动备份
        └── vmtools-export-*.json   # 导出文件（带时间戳）
```
