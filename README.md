# Venus Mc Tools (VMTools)

一个轻量实用的 Minecraft Fabric 客户端模组，核心功能是**路径点管理**和**快速传送**。通过向服务器发送自定义命令实现传送，适用于原版生存服务器。

> 支持 Minecraft 1.21.6、1.21.11 和 26.1 三版本

## ✨ 功能特性

### 🎯 路径点管理
- **快捷键打开**：默认按 `M` 键打开路径点管理界面
- **分组窗口**：每个分组是独立的可拖动小窗口，可放置在屏幕上任意位置
- **窗口尺寸自定义**：右边缘拖动调整宽度，底部边缘拖动调整高度，自定义尺寸持久保存
- **动态宽度**：窗口宽度根据分组名和路径点名长度自动适配
- **展开/折叠**：左键点击分组标题栏切换
- **滚动条**：路径点较多时自动显示滚动条，支持鼠标滚轮
- **路径点拖拽排序**：每个路径点前有拖拽手柄（⋮⋮），长按拖动排序
- **路径点 CRUD**：添加、编辑、删除、复制路径点（右键菜单操作）
- **分组管理**：创建、编辑、删除分组（右键菜单操作）
- **分组动态置顶**：点击分组自动显示到最上层
- **颜色标记**：9 种颜色可选，路径点和分组均可独立设置
- **左键点击路径点名称直接传送**
- **搜索过滤**：按名称或命令搜索路径点，隐藏无匹配的分组
- **Toast 通知**：操作反馈提示（多行显示，右上角弹出）

### 🛡️ 逃逸小工具
按 `M` 键打开界面后，通过顶部 Tab 栏切换到「逃逸小工具」页面。包含两个独立小工具：

#### 虚空逃逸
- **功能**：实时监控玩家 Y 坐标，低于阈值时自动执行命令
- **默认阈值**：Y = -70
- **默认命令**：`/ehomes home`
- **防重复触发**：触发一次后标记，回到安全高度才重置
- **自动确认**：可开启自动发送确认命令
- **可拖动窗口**：独立窗口，可拖动、调大小、折叠/展开
- **实时状态**：显示当前位置和触发状态
- **触发日志**：记录最近触发事件

#### 血量监控
- **功能**：实时监控玩家血量，低于阈值时自动执行命令
- **默认阈值**：10（半颗心，满血 20）
- **默认命令**：`/home`
- **防重复触发**：血量恢复后重置
- **自动确认**：可开启自动发送确认命令
- **可拖动窗口**：独立窗口，可拖动、调大小、折叠/展开
- **实时状态**：显示当前血量和状态
- **触发日志**：记录最近触发事件

### 📥 导入导出
- **导出**：将路径点导出为 JSON 文件，导出后自动打开目录
- **导入**：弹出系统文件选择窗口，直接选择 JSON 文件导入
- **导入预览**：导入前显示分组和路径点数量预览
- **导入模式**：支持追加导入和替换全部两种模式

### ⚙️ 设置页面
- **统一设置入口**：底部「设置」按钮
- **自动确认传送**：全局开关，传送后自动执行确认命令
- **自定义确认命令**：默认 `res tpconfirm`，可改成任意命令
- **可调延迟**：默认 200ms，范围 100-2000ms

### ✏️ 命令补全
- 新建/编辑路径点时，命令输入框自动补全
- 支持命令：`res tp`、`warp`、`home`、`tpa`、`tpask`、`ehomes home`
- 输入首字母自动匹配，支持多个匹配结果
- **鼠标点击**选择 / **键盘 ↑↓** 切换 / **Tab 或 Enter** 确认

### ⌨️ 快捷键
| 快捷键 | 功能 |
|--------|------|
| `M` | 打开路径点管理 / 逃逸小工具界面 |

快捷键可在游戏设置中修改。

## 📦 安装

1. 确保已安装 [Fabric Loader](https://fabricmc.net/)
2. 下载 [Fabric API](https://modrinth.com/mod/fabric-api) 并放入 `mods` 文件夹
3. 下载对应 Minecraft 版本的 VMTools JAR 并放入 `mods` 文件夹
4. 启动游戏

### 版本选择

| Minecraft 版本 | 下载文件 | Java 版本 | 分支 |
|---------------|---------|----------|------|
| 1.21.6 | `vmtools-mc1.21.6-2.1.1.jar` | 21+ | `mc-1.21.6` |
| 1.21.11 | `vmtools-mc1.21.11-2.1.1.jar` | 21+ | `main` |
| 26.1 | `vmtools-mc26.1-2.1.1.jar` | 25+ | `mc-26.1` |

## 🎮 使用方法

### 路径点管理
1. 按 `M` 键打开界面
2. 点击底部「+ 路径点」或「+ 分组」按钮
3. 左键点击路径点名称直接传送
4. 右键路径点/分组进行编辑、复制、删除

### 逃逸小工具
1. 按 `M` 键打开界面，点击顶部「逃逸小工具」Tab
2. 点击「虚空逃逸」或「血量监控」标题栏展开/折叠
3. 勾选「启用」开关，设置阈值和命令
4. 窗口可拖动位置、调整大小，状态自动保存

### 拖动窗口
- 左键长按标题栏拖动
- 短按（移动 < 5px）切换展开/折叠
- 鼠标靠近边框拖动调整宽度/高度

## 📁 数据文件

```
.minecraft/config/vmtools/
├── waypoints.json          # 路径点数据
├── waypoints.backup.json   # 自动备份
├── ui_state.json           # 路径点 UI 状态
├── auto_escape.json        # 虚空逃逸配置
├── health_monitor.json     # 血量监控配置
├── tool_windows.json       # 逃逸小工具窗口状态
└── vmtools-export-*.json   # 导出文件
```
