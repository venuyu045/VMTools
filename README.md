# Venus Mc Tools (VMTools)

一个轻量实用的 Minecraft Fabric 客户端模组，核心功能是**路径点管理**和**快速传送**。通过向服务器发送自定义命令实现传送，适用于原版生存服务器。

> 支持 Minecraft 1.21.6、1.21.11 和 26.1 三版本

## ✨ 功能特性

### 🎯 路径点管理
- **快捷键打开**：默认按 `M` 键打开路径点管理界面
- **分组窗口**：每个分组是独立的可拖动小窗口，可放置在屏幕上任意位置
- **窗口尺寸自定义**：右边缘拖动调整宽度，底部边缘拖动调整高度，自定义尺寸持久保存
- **动态宽度**：窗口宽度根据分组名和路径点名长度自动适配
- **展开/折叠**：左键点击分组标题栏切换
- **滚动条**：路径点较多时自动显示滚动条，支持鼠标滚轮
- **路径点拖拽排序**：每个路径点前有拖拽手柄（⋮⋮），长按拖动排序
- **路径点 CRUD**：添加、编辑、删除、复制路径点（右键菜单操作）
- **分组管理**：创建、编辑、删除分组（右键菜单操作）
- **分组动态置顶**：点击分组自动显示到最上层
- **颜色标记**：9 种颜色可选，路径点和分组均可独立设置
- **左键点击路径点名称直接传送**
- **搜索过滤**：按名称或命令搜索路径点，隐藏无匹配的分组
- **Toast 通知**：操作反馈提示（多行显示，右上角弹出）

### 📥 导入导出
- **导出**：将路径点导出为 JSON 文件，导出后自动打开目录
- **导入**：弹出系统文件选择窗口，直接选择 JSON 文件导入
- **导入预览**：导入前显示分组和路径点数量预览
- **导入模式**：支持追加导入和替换全部两种模式

### ⚙️ 设置页面
- **统一设置入口**：底部「设置」按钮
- **自动确认传送**：全局开关，传送后自动执行确认命令
- **自定义确认命令**：默认 `res tpconfirm`，可改成任意命令
- **可调延迟**：默认 200ms，范围 100-2000ms

### 🔄 自动确认传送
- **全局开关**：在设置页面统一控制
- **路径点级别控制**：每个路径点可独立设置开/关（编辑时勾选「自动确认」）
- **三级优先级**：路径点设置 > 全局设置
- **悬浮提示**：鼠标悬停显示「请务必保证目的地安全」

### ✏️ 命令补全
- 新建/编辑路径点时，命令输入框自动补全
- 支持命令：`res tp`、`warp`、`home`、`tpa`、`tpask`、`ehomes home`
- 输入首字母自动匹配，支持多个匹配结果
- **鼠标点击**选择 / **键盘 ↑↓** 切换 / **Tab 或 Enter** 确认

### ⌨️ 快捷键
| 快捷键 | 功能 |
|--------|------|
| `M` | 打开路径点管理界面 |

快捷键可在游戏设置中修改。

## 📦 安装

1. 确保已安装 [Fabric Loader](https://fabricmc.net/)
2. 下载 [Fabric API](https://modrinth.com/mod/fabric-api) 并放入 `mods` 文件夹
3. 下载对应 Minecraft 版本的 VMTools JAR 并放入 `mods` 文件夹
4. 启动游戏

### 版本选择

| Minecraft 版本 | 下载文件 | Java 版本 | 分支 |
|---------------|---------|----------|------|
| 1.21.6 | `vmtools-mc1.21.6-1.15.6.jar` | 21+ | `mc-1.21.6` |
| 1.21.11 | `vmtools-mc1.21.11-1.15.6.jar` | 21+ | `main` |
| 26.1 | `vmtools-mc26.1-1.15.6.jar` | 25+ | `mc-26.1` |

## 🎮 使用方法

### 添加路径点
1. 按 `M` 键打开路径点管理界面
2. 点击底部的「+ 路径点」按钮
3. 输入名称和传送命令（如 `/res tp main_city`）
4. 选择所属分组和颜色
5. 点击「添加路径点」保存

### 添加分组
1. 按 `M` 键打开路径点管理界面
2. 点击底部的「+ 分组」按钮
3. 输入分组名称，选择颜色
4. 点击「添加分组」保存

### 拖动窗口
- 左键长按分组标题栏，拖动到屏幕任意位置
- 短按（移动距离 < 5px）视为点击，切换展开/折叠

### 调整窗口大小
- **宽度**：鼠标靠近窗口右边缘，边框高亮后长按拖动
- **高度**：鼠标靠近窗口底部边缘，边框高亮后长按拖动
- 自定义尺寸自动保存，下次打开保持

### 路径点排序
- 左键长按路径点前的拖拽手柄（⋮⋮），上下拖动排序
- 排序自动保存，下次打开保持

### 分组置顶
- 点击任意分组（左键或右键），该分组显示到最上层
- 层级顺序自动保存

### 传送
- 左键点击路径点名称，客户端自动发送命令到服务器
- 如开启了自动确认，会延迟后自动发送确认命令

### 右键菜单
- **右键分组标题栏**：展开/折叠、编辑、删除
- **右键路径点**：编辑、复制、删除
- 删除操作需要二次确认，防止误触

## 📁 数据文件

```
.minecraft/config/vmtools/
├── waypoints.json          # 路径点数据（主文件）
├── waypoints.backup.json   # 自动备份（每次保存前生成）
├── ui_state.json           # UI 状态（窗口位置、展开、宽度、高度、层级）
└── vmtools-export-*.json   # 导出文件（带时间戳）
```

更新模组版本时，配置文件会自动保留。如主文件损坏，模组会自动从备份恢复。

## 📝 命令示例

| 服务器插件 | 命令格式 | 示例 |
|-----------|---------|------|
| Residence | `/res tp <名称>` | `/res tp main_city` |
| EssentialsX | `/home <名称>` | `/home base` |
| MyWarp | `/warp <名称>` | `/warp shop` |
| CMI | `/cmi warp <名称>` | `/cmi warp spawn` |

## 🎨 UI 特性

- **暗色主题**：Catppuccin 风格深色背景
- **可拖动窗口**：每个分组独立窗口，自由排列
- **可调整尺寸**：窗口宽度和高度均可拖动边框自定义
- **动态宽度**：窗口宽度自动适配内容长度
- **圆角卡片**：路径点使用圆角矩形容器
- **彩色标识**：9 种颜色可选
- **滚动条**：路径点列表超出高度时自动显示
- **Toast 通知**：右上角弹出，长文本自动换行
- **边框高亮**：鼠标靠近可拖拽边框时紫色高亮
